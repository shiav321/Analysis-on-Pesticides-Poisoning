Core Algorithm
The DRC-SML Approach
Dynamic Rule-based Classification with Supervised Machine Learning
A custom hybrid algorithm developed specifically for small clinical datasets with complex multi-class output patterns.
Rule-based layer — medical expert rules applied first to filter obvious cases and reduce noise
Supervised ML layer — trained classifier handles borderline and complex cases after rule filtering
Dynamic thresholding — adapts classification boundaries based on symptom pattern distributions
Result — 98% accuracy on 1,027 samples, outperforming standard ML approaches by a significant margin
